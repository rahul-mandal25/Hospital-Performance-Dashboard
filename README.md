🏥 Hospital Performance Dashboard

📌 Project Overview
This project is an interactive **Hospital Performance Dashboard** built using Power BI.  
It helps analyze hospital operations, patient data, and financial performance.

The dashboard provides insights into:
- Patient admissions
- Department performance
- Revenue & profit trends
- Patient satisfaction

🎯 Objectives
- Monitor hospital performance using key KPIs  
- Analyze patient distribution across departments  
- Track revenue, expenses, and profit trends  
- Improve decision-making using data visualization  

📊 Key Features
- 📌 Total Patients, Revenue, and Profit overview  
- 📌 Department-wise patient analysis  
- 📌 Monthly revenue trend analysis  
- 📌 Admission type distribution (Emergency vs Planned)  
- 📌 Age group analysis  
- 📌 Interactive filters (Gender, Department, Admission Type)  

 Tools & Technologies Used
- Power BI  
- Microsoft Excel  
- DAX (Data Analysis Expressions)  
- Data Visualization  

📈 Key Insights
- Emergency admissions are slightly higher than planned admissions  
- Revenue shows a declining trend in later months  
- Some departments have significantly higher patient load  
- Adult age group contributes the highest number of patients  

## 🚀 How to Use
1. Download the `.pbix` file from this repository  
2. Open it using Power BI Desktop  
3. Interact with filters and visuals to explore insights
